# Birthday Quest (React + Vite)

Сайт-квест на день рождения с этапами:
1. Приветствие
2. Проверка (время и дата рождения)
3. Пошаговые квесты (следующий открывается только после правильного ответа)
4. Финальное поздравление

## Стек
- React + Vite
- GitHub Actions (автосборка)
- GitHub Pages (хостинг)

## Автодеплой без локального npm

Проект уже настроен на автодеплой через workflow:
- `.github/workflows/deploy-pages.yml`

Что делать:
1. Запушь код в ветку `main`.
2. Открой GitHub: `Settings` -> `Pages`.
3. В `Source` выбери `GitHub Actions`.
4. После пуша откроется workflow `Deploy to GitHub Pages`, и сайт опубликуется автоматически.

URL будет вида:
`https://<твой-логин>.github.io/<имя-репозитория>/`

## Локальный запуск (опционально)

Нужен только если хочешь тестировать сайт у себя на ПК.

```bash
npm install
npm run dev
```

## Как менять квесты и ответы

Все данные квеста находятся в `src/data/quests.js`.

### Проверка входа
```js
export const BIRTHDAY_GATE = {
  time: '00:00',
  date: '8-4-2005'
}
```

### Квесты
```js
export const QUESTS = [
  {
    id: 'q1',
    title: 'Квест 1',
    question: 'Ты кто?',
    answer: 'илья'
  },
  {
    id: 'q2',
    title: 'Квест 2',
    question: 'Сколько квестов ты прошел?',
    answer: '1'
  }
]
```

Чтобы добавить квест, добавь новый объект в `QUESTS`.

## Важно

- Базовый путь (`base`) в `vite.config.js` настроен автоматически для GitHub Pages.
- Для автодеплоя локальная установка Node/npm не требуется: сборка идет на GitHub runner.
